# vue_auth

## .env sample

Create a .env file on the project root with the content:

```
VUE_APP_API_URL=http://localhost:3000
VUE_APP_TIME_TO_REFRESH_TOKEN_BEFORE_ACCESS_TOKEN_EXP_IN_MILLI=15000
```
